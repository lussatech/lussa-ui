The Frameworks

Secara singkat lussa UI ini merupakan framework yang beriorentasi pada Aksen dan Bentuk. berawal dari greget framework UI saat ini mengharuskan seorang front-end developer memahami banyak komponen unik masing-masing framework dengan semantik yang sulit untuk dipahami, dan mengharuskan developer mengikuti frame kaku tersebut membuat developer awam kesulitan untuk membuat desain yang diinginkan.

Kurang lebih begitu-lah, kedepannya framework ini akan dibuat se-fleksible dan se-versatile mungkin, apalagi dengan datangnya teknologi baru yang memecahkan banyak masalah-masalah klasik kayak clearfix, wrapping, vertical alignment dkk. yaitu ES6, web-component, angular 2.0, Flexbox dkk.
