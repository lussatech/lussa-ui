lussa ui dibikin pake node js environment,

Prerequisite

Lussa UI dibuat untuk memenuhi kebutuhan development web masa kini yang berbasis MEAN - stacks. Jika kamu adalah developer MEAN - stack seharusnya semua prerequisite dibawah ini sudah di-install.

git
Pastikan git sudah diinstall dan sudah berada didalam path system. Kalo belum ada, rekomendasi kami bisa di-download langsung di official websitenya langsung git-scm.com {% highlight shell %} # konfirmasi installasi git git -v {% endhighlight %}
node js
Node js merupakan komponen penting, karena lussa ui dibangun diatas lingkungan ini {% highlight shell %} # konfirmasi installasi nodejs node -v {% endhighlight %}
grunt client
Grunt client merupakan task-runner yang akan menjalankan banyak fungsi, mulai dalam compiling hingga membuat dokumentasi {% highlight shell %} # installasi npm install grunt-cli -g {% endhighlight %}
bower
Bower digunakan untuk menginstall beberapa dependency front-end untuk proses testing dan dokumentasi tarsius UI, komponen penting seperti jQuery dan angular diinstall melalui package manager ini {% highlight shell %} # installasi npm install bower -g {% endhighlight %}

Installasi Menggunakan Node Package Manager a.k.a NPM

Secara resmi, lussa ui belum ada di npm, untuk sementara installasi dengan NPM bisa dilakukan dengan menggunakan alamat repository dari lussa ui

{% highlight shell %} # installasi dengan menggunakan npm npm install http://git.lussa.net/tarsius/tarsius-ui.git --save-dev {% endhighlight %}

Installasi Menggunakan Bower

Sama halnya dengan NPM secara resmi, lussa ui belum ada di bower, untuk sementara installasi bower bisa dilakukan dengan menggunakan alamat repository dari lussa ui. Dengan menggunakan bower, kamu bisa lebih rapih dalam mengatur dependency dari aplikasi yang bakal develop.

{% highlight shell %} # installasi dengan menggunakan bower bower install http://git.lussa.net/tarsius/tarsius-ui.git --save {% endhighlight %}

Build Langsung Dari Source

Kalo kamu pengen merubah beberapa parameter dari source, atau cuma butuh beberapa component dari framework ini, kamu bisa build manual dari source, dan atur file less / js utamanya.

Download source bisa langsung download snapshot master / branch development, atau bisa di clone pake git

{% highlight shell %} # cloning pake git git clone http://git.lussa.net/tarsius/tarsius-ui.git # download langsung dari snapshot wget http://git.lussa.net/tarsius/tarsius-ui/repository/archive.zip {% endhighlight %}

Jangan lupa install beberapa development library buat ngebuild sourcenya, cukup pake npm aja.

{% highlight shell %} # install tools development npm install ...... {% endhighlight %}

kalo kamu juga butuh dokumentasinya, kamu juga perlu nginstall dependency buat fron-endnya pake bower

{% highlight shell %} # install dependency buat dokumentasi bower install ...... {% endhighlight %}

kalo kamu sudah punya grunt-client di system, pastikan gruntnya sudah ada di path system, atau nodejs shell.

{% highlight shell %} # konfirmasi grunt grunt -v grunt-cli: The grunt command line interface. (v0.1.13) {% endhighlight %}

Kalo udah cukup jalananin task untuk build source, prosesnya sudah dipermudah pake grunt, kalo mau agak diulik cukup buka aja file gruntfile.js nya

{% highlight shell %} # build semua grunt dist # build css aja grunt css-dist # build js aja grunt js-dist {% endhighlight %}
