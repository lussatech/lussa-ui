Overview

Intro

Component disini adalah elemen-elemen non standar html, maupun elemen-elemen standar yang dirasa perlu di-extend untuk meningkatkan aspek user experience maupun compability element tersebut.

Bingung? oke saya kasih contoh kasus. Misal elemen .input[type=date] pada browser chrome jika di klik akan memberikan kalendar yang dapat dipilih, tetapi di browser firefox muncul tampilan dengan format berbeda. Hal-hal seperti ini dirasa perlu di ganti dengan Component.

Kemudian ada elemen seperti input autocomplete, input tag yang secara default tidak ada pada html standar dan dirasa perlu dibuat direktif khusus, disertakan juga pada lussa UI.

Selain itu elemen-elemen seperti slider, atau loading-bar yang digunakan dibanyak aplikasi. Agar mempercepat proses prototyping dirasa perlu dibuat.

Dependency

Modul lussa UI hanya bisa bekerja pada lingkungan aplikasi yang berbasis angular, selain angular lussa UI juga bergantung terhadap modul-modul ini. Kamu bisa menginstallnya menggunakan bower lalu disertakan dalam aplikasi yang kamu buat.

Jika kamu menginstall lussa-ui menggunakan bower, kamu cukup menyertakan file dependecy saja, tidak perlu menginstall dependency secara manual.

{% highlight json %} "dependencies": { "jquery": ">=1.7.0", "jquery-easing-original": "1.3.2", "jquery-hoverIntent": "~1.8.1", "underscore": "~1.8.2", "angular": "~1.3.15", "angular-animate": "~1.3.15", "angular-sanitize": "~1.3.15", "underscore.string": "~3.0.3" } {% endhighlight %}

Penggunaan

Sebelum dapat menggunakan custom component dari tarsius-ui kamu harus menyertakan file lussa-ui.js dan menyertakan modul lussa.ui ke dalam dependency aplikasi angular kamu.

{% highlight html %} ... {% endhighlight %} {% highlight js %} var app = angular.module('YourFunkyAppName', [ ... 'lussa.ui' ]); {% endhighlight %}
