Tooltip

Browser sebenarnya sudah menyediakan default Tooltip dengan menambahkan atribut title. Tooltip pada lussa-ui berfungsi untuk menambah fungsi & aksen warna pada tooltip browser. Tampilan dapat ditambahkan di kustom dengan tambahan kelas, maupun dapat diiisi dengan elemen html lain.

ada 3 versi tooltip yaitu tooltip, tooltip-template dan tooltip-html

  • tooltip cuma bisa nerima teks, semua html akan di-escape
  • tooltip-template dapat menerima url / lokasi template yang akan digunakan oleh tooltip
  • tooltip-html memenerima konten HTML dan akan di-render. User bertanggung jawab terhadap keamanan konten yang bakal di load di DOM
  • tooltip-html-unsafe versi deperecated dari tooltip-html

  • tampilan
  • <html>
  • penggunaan
  • theming

Popover

Popover merupakan varian tooltip yang menyerupai panel. Bisa digunakan untuk memuat informasi tambahan yang cukup banyak.

ada 2 versi popover yaitu popover dan popover-template

  • popover cuma bisa nerima teks, semua html akan di-escape
  • popover-template dapat menerima url / lokasi template yang akan digunakan oleh bagian badan dari popover

  • tampilan
  • <html>
  • penggunaan
  • theming
