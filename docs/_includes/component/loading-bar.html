Loading Bar

sebuah loading bar yang bekerja secara otomatis dengan membaca request XHR aplikasi angular kamu.

Setup

Seperti halnya toast, loading bar dapat diatur langsung di fasa konfigurasi aplikasi kamu. loading bar juga memiliki beberapa event dan service yang dapat digunakan didalam controller kamu.

  1. Atur parameter loading bar pada fase konfigurasi
    2. {% highlight js %} angular.config(['loadingBar' ,function(loadingBar){ loadingbar.latencyThreshold = 100; loadingbar.startSize = 0.02; loadingbar.parentSelector = 'body'; }]); {% endhighlight %}
  2. You good to go!

Service

Loading bar sendiri, dipersenjatai 5 method yang dapat digunakan untuk memanipulasi loading bar yang dapat kamu panggil didalam controller aplikasi kamu.

  • tampilan
  • controller

loadingBar.status {( (docs.loadingBar.status() * 100) + ' %' )}

{% highlight js %} app.controller('loadingbarCtrl' ,['loadingBar',function(loadingBar){ // start the bar loadingBar.start(); // set to 0.32 loadingBar.set(0.32); // 0 - 1 loadingBar.inc(); // increment // finish the bar & destory loadingBar.complete(); // get status var status = loadingBar.status; }]); {% endhighlight %}

Event

Loading-bar memancarkan (broadcast) beberapa event

loadingBar:loading
di broadcast ketika loading bar menerima XHR request yg belum di cache
loadingBar:loaded
di broadcast ketika loading bar menerima XHR resddond, baik sukses maupun gagal
loadingBar:completed
di broadcast ketika semua antrian XHR task telah selesai di eksekusi
loadingBar:started
di broadcast ketika memulai sebuah antrian XHR task
