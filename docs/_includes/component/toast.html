Setup

sebelum kamu bisa menggunakan toast, kamu harus melakukan beberapa hal penting.

  1. Masukan direktif toast sebelum tag body ditutup.
    2. {% highlight html %} ... {% endhighlight %}
  2. Bila diperlukan, lakukan beberapa konfigurasi pada fase config
    3. {% highlight js %} angular.config(['toast', function(toast){ toast.className: 'info', toast.dismissOnTimeout: true, toast.timeout: 5000, toast.toast.dismissButton: true, toast.dismissButtonHtml: '', toast.dismissOnClick: true, toast.compileContent: false, toast.horizontalPosition: 'right', // right, center, left toast.verticalPosition: 'top', // top, bottom, toast.maxNumber: 0 }]); {% endhighlight %}
  3. Panggil method toast.create didalam controller, untuk menguji toast.
    4. {% highlight js %} toast.create({ content: "howdy!" className: "success" }); // atau bisa juga toast.create("howdy!"); {% endhighlight %}

Pengaturan / Config

Toast sendiri terdiri dari 2 elemen utama, yaitu toast container dan toast message. Beberapa konfigurasi umum dapat dilakukan pada fase config. Sedangkan untuk menampilkan toast message kamu bisa mengatur konten, kelas dan beberapa konfigurasi unik per pesan.

  • global config
  • message config
{% highlight js %} className: 'info', dismissOnTimeout: true, timeout: 5000, dismissButton: true, dismissButtonHtml: '', dismissOnClick: true, compileContent: false, horizontalPosition: 'right', // right, center, left verticalPosition: 'top', // top, bottom, maxNumber: 0 {% endhighlight %}
{% highlight js %} toast.create({ content: "{string}", [id: "{float}"], [className: "{string}"], [dismissOnTimeout: "{boolean}"], [timeout: "{integer}"], [dismissButton: "{boolean}"], [dismissButtonHtml: "{string}"], [dismissOnClick: "{boolean}"], [compileContent: "{boolean}"] }); {% endhighlight %}
