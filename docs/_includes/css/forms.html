Inline Form

Kalau kamu ingin form dengan tampilan sebaris (inline) dari kiri ke kanan, kamu cukup menambahkan class .form-inline pada tag <form>. Pada layar mobile, tampilan ini akan secara responsive stacking ke bawah

Horizontal Form

Tambahkan class .form-horizontal untuk membuat layout form dengan label dan input saling berseblahan, seperti halnya pada horizontal description list. Pada dasarnya horizontal form memanfaat grid untuk membentuk layoutnya, disini akan diberikan contoh untuk membuat layout form horizontal.

Jangan lupa atur ukuran grid untuk kolom label dan kolom .form-control.

Keadaan / State dari From Control

Form control memiliki beberapa keadaan / status standar yang merepresentasikan konteks UX sebuah input. Misal ketika input dalam keadaan di fokus, di matikan (disabled) atau hanya bisa dibaca (readonly).

Tampilan disabled berlaku juga bila <fieldset> juga dalam state disabled. Jadi jika ingin mendisabled semua input secaram bersamaan cukup gunakan <fieldset disabled>
