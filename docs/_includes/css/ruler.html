Horizontal Ruler

Sesuai namanya, garis ini memisahkan elemen secara horizontal, merupakan elemen murni dari html <hr>, dapat juga menggunakan class .hr untuk elemen lain. Secara default, elemen hr memiliki margin atas dan bawah sebesar computed line height, atau sekita 17px tergatung konfigurasi @font-size-base
