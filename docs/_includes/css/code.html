Code

Tipografi khusus untuk kode program

Inline

Kamu bisa pake tag <code> secara inline untuk konteks teks yang berhubungan dengan kode/syntax, misal tag html nama fungsi dll.

  • tampilan
  • <html>
For example, <section> should be wrapped as inline.
{% highlight html %} For example, <section> should be wrapped as inline. {% endhighlight %}

User Input

Untuk teks yang menunjukan input dari user (keyboard), kamu dapat menggunakan tag <kbd> secara inline.

  • tampilan
  • <html>
To switch directories, type cd followed by the name of the directory.
To edit settings, press ctrl + ,
{% highlight html %} To switch directories, type cd followed by the name of the directory.
To edit settings, press ctrl + , {% endhighlight %}

Blok Kode

Gunakan tag <pre> untuk menampilkan paragraf untuk source-code

  • tampilan
  • <html>
<p>Sample text here...</p>
{% highlight html %} 
<p>Sample text here...</p>
{% endhighlight %}

Variable / Fungsi

Gunakan tag <var> untuk menandakan sebuah teks sebagai variable / fungsi

  • tampilan
  • <html>
y = mx + b
{% highlight html %} y = mx + b {% endhighlight %}

Sample Output

Gunakan tag <samp> untuk menunjukan output dari sebuah program

  • tampilan
  • <html>
This text is meant to be treated as sample output from a computer program.
{% highlight html %} This text is meant to be treated as sample output from a computer program. {% endhighlight %}
