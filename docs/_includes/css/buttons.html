Sesuai namanya button group, merupakan elemen tambahan pada lussa UI yang berfungsi untuk mengelompokan tombol menjadi suatu blok khusus. Biasa digunakan pada pagination, atau bisa juga dipakai sebagai action button.

Justified

.button-group biasa bersifat inline, artinnya apabila ada elemen setelahnya, akan berada disamping .button-group. Hal ini juga menyebabkan kita bisa menggunakan tombol dengan panjang berbeda-beda. Jika ingin button group membloking 1 line, tanpa perlu mengatur panjang masing-masing tombol, gunakan kelas button-group-justified.

Untuk elemen <button>, kamu harus menggunakan kelas .button-group sebagai pembungkus

Button State

Berdasakan state-nya (keadaan), lussa ui memberikan aksen yang berbeda-beda. Ada 2 aksen tambahan berdasarkan state tombol, active state dan disabled state. Secara default css mengatur active state dengan menggunakan selector :active, dan akan ditampilkan ketika tombol dalam keadaan ditekan. Tetapi kita dapat memaksakan tampilan active dengan menambahkan class .active pada button tag manapun.
