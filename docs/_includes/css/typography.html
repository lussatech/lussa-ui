Inline text elements

kita bisa menambahkan beberapa elemen inline yang biasa ada didalam sebuah teks, untuk merepresentasikan konteks sebuah tulisan. Element-element tersebut antara lain

Marked Text
Biasa digunakan untuk men-stabilo / marking sebuah teks dengan tag <mark>
Deleted Text
Biasa digunakan untuk menunujukan teks yang sudah tidak ada lagi / dihapus dengan tag <del>
Striketrough Text
Biasa digunakan untuk menunujukan teks yang sudah tidak relevan lagi dengan tag <s>
Inserted Text
Biasa digunakan untuk menunjukan penambahan dengan tag <ins>
Underline Text
Untuk meng-underline sebuah teks cukup dengan tag <u>
Small Text
Biasa digunakan untuk menandai footnote / alternative teks dengan tag <small> atau class .small
Bold
Untuk element yang berukuran tebal bisa menggunakan <strong>
Italics
Untuk memiringkan teks bisa dengan tag <em>, biasa dipakai untuk kalimat / istilah asing
