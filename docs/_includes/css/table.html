Tabel Umum / Basic

Untuk tabel umum / biasa bisa menggunakan tag <table> ditambah dengan class .table. Penggunaan class .table bertujuan untuk mengisolasi style untuk table biasa. Karena kebanyakan tag table digunakan untuk layouting / tampilan direktif seperti date-picker dan cart system.

  • tampilan
  • <html>
Optional table caption.
# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter
{% highlight html %} ...
Optional table caption.
# ...
1 ...
{% endhighlight %}

Tabel Belang-belang / Zebra

Untuk tampilan belang-belang cukup tambahkan class .table-stripped di table, otomatis semua baris yang ada di dalam <tbody> jadi belang-belang

  • tampilan
  • <html>
# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter
{% highlight html %} ...
# ...
1 ...
{% endhighlight %}

Tabel Responsive

Untuk membuat tabel menjadi responsive kamu cukup membungkus tabel dengan container berkelas .table-responsive. Container secara otomatis akan dapat discroll ketika konten table didalamnya lebih lebar dari screen device.

  • tampilan
  • <html>
# Table heading Table heading Table heading Table heading Table heading Table heading
1 Table cell Table cell Table cell Table cell Table cell Table cell
2 Table cell Table cell Table cell Table cell Table cell Table cell
3 Table cell Table cell Table cell Table cell Table cell Table cell
{% highlight html %}
...
