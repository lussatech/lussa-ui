Kelas Komposit

kelas yang bila dipadukan akan menghasilkan elemen baru yang unik, bisa diimplementasi ke elemen lain yang sudah memiliki tampilan default.

Aksen

Jika kamu amati, hampir tiap elemen seperti tombol dan garis pemisah memiliki variasi warna khusus. Sebenernya, hampir semua elemen html dapat memiliki variasi warna tersebut, yaitu dengan menggunakan kelas-kelas composite khusus aksen. Kelas-kelas aksen akan memberikan aksen warna pada text, background dan bg. Lussa UI sendiri mengklasifikan aksen menjadi 3 kelompok yaitu,

  • Berdasarkan Tingkat Kecerahan (by brightness & translucency)
    • neutral
    • dark
    • light
    • soft
  • Berdasarkan Kontras
    • vibrant
    • clear
  • Berdasarkan Konteks
    • primary
    • success
    • info
    • warning
    • error
    • muted

Responsive Ratio Class

Responsive ratio class, adalah kelas komposit yang berfungsi untuk membuat bentuk yang memiliki rasio ukuran yang tetap meskipun lebar elemen tersebut berubah. Kelas ini sangat berguna untuk membuat elemen-elemen yang bersifat interaktif.

Menggunakan kelas ini cukup mudah, kamu cukup menambahkan kelas .ratio-x-y, dimana x dan y secara berurutan adalah rasio lebar dan tinggi elemen. Didalam elemen ini kamu wajib memasukkan kelas .ratio-content untuk mengisi konten elemen ini.

Kombinasi rasio yang tersedia yaitu dari 1:1,1:2 ... 7:8 dengan tambahan rasio 16:9 / 9:16

Vertical Screen Class

Tinggi yang tersedia kelipatan 25, 20 dan 33

Pading & Margin

Padding dan Margin dapat diimplementasikan pada elemen dengan menambahkan class padding-x-y, dengan x adalah varian arah, dan y adalah varian ukuran. Untuk diimplementasikan ke semua arah, cukup gunakan padding-y. Varian arah yang tersedia yaitu top,bottom, left, right, vertical dan horizontal. Varian ukuran yang tersedia yaitu medium,small, large, xl, dan xs.

Untuk margin cukup gunakan class margin-x-y, variable x dan y sama dengan padding.

Alignment Class

Yaitu kelas yang berfungsi untuk memposisikan elemen didalam container. Ada 3 kelas yaitu, .pull-right, .pull-left dan .center-block.

Kamu perlu mendefinisikan lebar maksimum (max-width) untuk element .center-block agar elemen tersebut tidak blocking 100%;

Untuk menghindari keanehan tampilan, parent elemen harus menggunakan kelas .clearfix untuk menghindari perbedaan tinggi antar parent dengan element. Secara default class .grid-container dan .grid-fluid sudah memiliki menggunakan kelas .clearfix

  • tampilan
  • <html>
.pull-right
.pull-left
.center-block
max-width: 200px;

{% endhighlight %}
